# Mail.tm OTP Viewer

Cara pakai online (disarankan):
1. Upload folder ini ke GitHub.
2. Deploy ke Vercel, Netlify, atau GitHub Pages.
3. Buka website, login dengan email Mail.tm.

Catatan:
Jika dibuka langsung sebagai file HTML, browser dapat memblokir akses API (CORS).
Deploy ke hosting statis gratis akan menghindari masalah tersebut.
